# Mobile-Final-Project



### TRIVIA GAME



##### Team:



Kiran Wood - API developer

Nick Tam - Gameplay Programmer

Tyler Jablonski - System Developer

Julian D Silva - UI Developer



##### Main Features:



* Access to a free trivia API with multiple topics of pop culture
* Persistent score tracking of the user to constantly try to beat your best score
* Optimized systems to present a fluent and clear experience
* Fast hooking gameplay which appeals to the trivia gerne games



##### How to use:



1. Use the project to access the build
2. Pick the difficulty of the questions
3. Start answering questions until you get it wrong during the different rounds
